UPLOAD ALL FILES IN THIS FOLDER TO THE ROOT OF YOUR GITHUB REPOSITORY.

Required filenames:
- index.html
- manifest.json
- service-worker.js
- apple-touch-icon.png
- icon-192.png
- icon-512.png

After GitHub Pages updates:
1. Open the site in Safari.
2. Press Command + Option + R on Mac to force refresh.
3. Delete the old Daily Practice app from the Dock / Applications.
4. Add the website to the Dock again.

Do not rename these files and do not keep names such as index(7).html.
